Enterprise-grade corporate website built with **HTML5, Tailwind CSS, and Vanilla JavaScript** — representing a modern technology consulting firm specializing in **AI-driven automation and business system optimization**.

```bash
# Open in browser

# Open the website
https://herox21.github.io/Orion-Bits-Systems-Pvt-Ltd/
```

No build step required.
